# TYPO3 Extension `rlt3sitepackage`
This extension can be used for starting a new TYPO3 project.

It's quite simple, very flexible and extendable.

It contains HTML5, PHP, JavaScript and plain CSS.

## Installation
First TYPO3 has to be installed. Then install `rlt3sitepackage`:

`composer req resterland/rlt3sitepackage`

### Compatibility

- TYPO3 = 13
- PHP = 8.2 - 8.4

### Hint - Tip
If you are new to TYPO3 or searching for a more sophisticated extension to get
your project up and running, I recommend you start with [bootstrap_package]
(https://github.
com/benjaminkott/bootstrap_package) by Benjamin
Kott.
